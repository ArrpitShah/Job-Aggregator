# 🔍 Job Aggregator

A centralized job search platform that aggregates listings from multiple job portals — search, filter, and apply from one place.

> 🚧 **Status: Work in Progress** — Core features under active development.

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express
- **Data:** Web scraping / Job APIs
- **Database:** Supabase (PostgreSQL)

---

## ✨ Planned Features

- [ ] Aggregate jobs from LinkedIn, Naukri, Internshala
- [ ] Search & filter by role, location, salary
- [ ] One-click apply tracking
- [ ] Email alerts for new matching jobs
- [ ] User dashboard with saved jobs

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ArrpitShah/Job-Aggregator.git

# Install dependencies
cd Job-Aggregator
npm install

# Run the app
npm run dev
```
